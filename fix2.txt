아래 항목들은 아직 구현되지 않은 새 기능이야. 진행해줘.

1. 없음/0 intent 처리: "없음", "다 떨어짐", "발주 필요", "소진" 표현을 재고 0으로 처리. "거의 없음"→0.2, "조금 있음"→0.3, "반/절반/1/2"→0.5.

2. 동일 품목 재입력 시 확인 카드: 같은 세션에서 동일 품목 재입력 감지 시 "이미 입력했습니다 / 합쳐서 N봉으로 할까요? / 새로 N봉으로 바꿀까요?" 카드 1회 표시. 계산 결과 반드시 표시. 5초 무응답 시 자동 종료 (자동 저장 아님).

3. 발주 리스트 출력 개선: "오늘 발주 뭐 해야 돼?" 응답에 [발주 생성] [수정] [취소] 버튼 추가. [발주 생성] 누르면 order_requests에 INSERT.

4. 입력창 전체 탭 고정: 재고/발주 탭에서도 상단 입력창 항상 표시.

5. 설정 변경 카드: "브로콜리 거래처 바꿔줘" 같은 입력 감지 시 [거래처] [최소수량] [목표수량] 선택 카드 표시. 거래처 선택 시 기존 거래처 목록 버튼으로 표시.
