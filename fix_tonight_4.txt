index.html UI를 채팅 구조로 개편해줘.

현재 구조:
- 입력창 상단 고정
- 처리 결과 아래 표시
- "따라해 보세요" 카드 고정

변경 구조:
- 입력창 하단 고정 (ChatGPT/Claude처럼)
- 대화 내용이 위로 쌓이는 구조
- 처리 결과가 말풍선 형태로 표시
- ZeroKitchen 답변은 왼쪽, 사용자 입력은 오른쪽
- 첫 진입 시 "따라해 보세요" 가이드가 첫 메시지처럼 표시

세부 요구사항:
1. 입력창은 항상 화면 하단 고정 (scroll 영향 없음)
2. 확인 카드, 합산 카드, 하네스 카드는 입력창 바로 위에 표시
3. 처리 완료된 결과는 대화 히스토리로 위에 쌓임
4. 각 처리 결과 카드에 [되돌리기] 버튼 포함 (30초 이내 활성)
5. 재고/발주 탭 전환 시에도 입력창은 항상 하단 고정 유지
6. 모바일 최적화 (키보드 올라왔을 때 입력창 위치 유지)

기존 기능 (Rule Engine, Supabase 연동, 하네스 등) 모두 유지.

완료 후 git add index.html && git commit -m "Redesign UI to chat-style layout with bottom input bar" && git push
