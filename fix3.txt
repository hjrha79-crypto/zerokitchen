toggleMic 함수 안에서 _recognition 설정 부분 수정.

변경 전:
  _recognition.continuous = false;

변경 후:
  _recognition.continuous = true;

그리고 onresult 핸들러도 수정해야 함.
continuous = true일 때는 결과가 누적되므로, 마이크가 수동으로 꺼질 때만 최종 텍스트를 처리해야 함.

기존 onresult:
  _recognition.onresult = (e) => {
    const inp = document.getElementById('nlInput');
    let final = '', interim = '';
    for (let i = 0; i < e.results.length; i++) {
      if (e.results[i].isFinal) final += e.results[i][0].transcript;
      else interim += e.results[i][0].transcript;
    }
    inp.value = final || interim;
    if (final) {
      _stopMic();
      setTimeout(() => handleNaturalInput(), 200);
    }
  };

변경 후:
  _recognition.onresult = (e) => {
    const inp = document.getElementById('nlInput');
    let final = '', interim = '';
    for (let i = 0; i < e.results.length; i++) {
      if (e.results[i].isFinal) final += e.results[i][0].transcript;
      else interim += e.results[i][0].transcript;
    }
    inp.value = final || interim;
    // continuous 모드: 자동 전송 안 함. 카운트다운 종료 또는 마이크 버튼 탭으로만 전송.
  };

그리고 _stopMic 함수에서 마이크 종료 시 입력창에 텍스트가 있으면 자동 전송하도록 수정.

기존 _stopMic:
function _stopMic() {
  if (_recognition) {
    try { _recognition.stop(); } catch(e) {}
  }
  _stopMicUI();
}

변경 후:
function _stopMic() {
  if (_recognition) {
    try { _recognition.stop(); } catch(e) {}
  }
  _stopMicUI();
  // 마이크 종료 시 입력창에 내용 있으면 자동 전송
  const inp = document.getElementById('nlInput');
  if (inp && inp.value.trim()) {
    setTimeout(() => handleNaturalInput(), 300);
  }
}
