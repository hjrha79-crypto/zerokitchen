index.html 기능 4개 수정해줘.

1. 처리 결과 [되돌리기] 버튼 추가
처리 결과 카드 하단에 [되돌리기] 버튼 추가.
클릭 시 해당 처리 이전 재고값으로 복원.
처리 후 30초 이내에만 활성화, 이후 자동 비활성 (버튼 회색 처리).
되돌리기 완료 시 "되돌렸습니다" 토스트 표시.
되돌린 후에도 다시 [되돌리기] 가능 (이전 상태로 복원).

2. 재고 탭에서 입력 시 확인 카드 표시
현재 재고 탭에서 입력하면 확인 없이 바로 처리됨.
입력 탭과 동일하게 확인·실행 카드 표시 후 처리하도록 수정.

3. 발주 탭 수량 표시 방식 수정
현재 "3박스 → 2박스" 표시가 직관적이지 않음.
변경: "현재 1박스 / 발주 2박스" 형태로 표시.
또는 "발주 수량: 2박스" 만 표시.

4. 발주 탭 거래처 변경 가능하게
현재 발주 탭에서 거래처 클릭 시 즉시 변경됨.
변경: 거래처 클릭 시 선택 카드 표시 후 확인 받고 변경.
카드: "칠리소스 거래처를 변경합니다 / [쿠팡] [쿠치나푸드] [기존거래처목록] / [취소]"
