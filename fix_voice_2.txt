api 폴더를 만들고 그 안에 transcribe.js 파일 생성:

그리고 api\transcribe.js 파일을 새로 만들어서 아래 내용을 넣기:

=== api/transcribe.js 전체 내용 ===

export const config = { runtime: 'edge' };

export default async function handler(req) {
  if (req.method !== 'POST') {
    return new Response(JSON.stringify({ error: 'Method not allowed' }), {
      status: 405,
      headers: { 'Content-Type': 'application/json' }
    });
  }

  const apiKey = process.env.OPENAI_API_KEY;
  if (!apiKey) {
    return new Response(JSON.stringify({ error: 'API key not configured' }), {
      status: 500,
      headers: { 'Content-Type': 'application/json' }
    });
  }

  try {
    // 클라이언트에서 보낸 FormData를 그대로 OpenAI로 전달
    const formData = await req.formData();

    // model과 language 강제 설정
    formData.set('model', 'gpt-4o-mini-transcribe');
    formData.set('language', 'ko');

    const response = await fetch('https://api.openai.com/v1/audio/transcriptions', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer ' + apiKey
      },
      body: formData
    });

    const data = await response.json();

    if (!response.ok) {
      return new Response(JSON.stringify({ error: data.error?.message || 'OpenAI API error' }), {
        status: response.status,
        headers: { 'Content-Type': 'application/json' }
      });
    }

    return new Response(JSON.stringify(data), {
      status: 200,
      headers: { 'Content-Type': 'application/json' }
    });
  } catch (e) {
    return new Response(JSON.stringify({ error: e.message }), {
      status: 500,
      headers: { 'Content-Type': 'application/json' }
    });
  }
}

=== 파일 끝 ===

수정 2: index.html — API 호출을 프록시로 변경
━━━━━━━━━━━━━━━━━━━━━

index.html에서 OPENAI_API_KEY 상수 제거 (이미 빈 문자열이지만 깔끔하게):

현재:
const OPENAI_API_KEY='';

이 줄을 삭제.

그리고 _transcribeAudio 함수 안에서 fetch 호출 부분을 수정:

현재:
    const formData = new FormData();
    formData.append('file', blob, 'recording.' + ext);
    formData.append('model', 'gpt-4o-mini-transcribe');
    formData.append('language', 'ko');

    const response = await fetch('https://api.openai.com/v1/audio/transcriptions', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer ' + OPENAI_API_KEY
      },
      body: formData
    });

변경 후:
    const formData = new FormData();
    formData.append('file', blob, 'recording.' + ext);

    const response = await fetch('/api/transcribe', {
      method: 'POST',
      body: formData
    });

변경 포인트:
- URL: api.openai.com → /api/transcribe (Vercel 프록시)
- headers: Authorization 헤더 제거 (서버에서 처리)
- model, language: 서버에서 강제 설정하므로 클라이언트에서 불필요
