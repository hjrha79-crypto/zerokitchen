품목명 수정 시 중복 감지 → 합치기 카드

재고 탭에서 품목명 수정 완료 시 동일한 이름의 품목이 이미 존재하면 합치기 카드 표시.
예: "훅 살라미" → "쿡 살라미"로 수정 시 기존 "쿡 살라미"(8개)가 있으면:

카드 문구:
"쿡 살라미"가 이미 있습니다.
[합치기 (8+5=13개)] [따로 유지] [취소]

합치기 선택 시: 두 품목 재고 합산 후 하나로 통합, 나머지 삭제.
따로 유지 선택 시: 두 품목 별도 유지.

완료 후 git add index.html && git commit -m "Add duplicate detection on item rename" && git push
