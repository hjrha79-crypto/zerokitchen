index.html 버그 6개 수정해줘.

1. "발주" 또는 "재고" 단어 단독 입력 시 Cannot set properties of null (innerHTML) 오류. 해당 탭으로 이동하거나 안내 메시지 표시.

2. 재고 0인 품목이 발주 목록에 안 뜨는 버그. min_stock 이하이면 발주 목록에 표시 (0 포함).

3. "치즈 3봉 입고 토마토소스 재고 얼마야" 입력 시 전체 재고 목록이 출력되는 오류. 입고 처리와 재고 조회를 분리해서 각각 처리.

4. "쪽파 2단 입고"처럼 미등록 품목 입력 시 유사 품목 카드 없이 쪽파와 입고가 별도 품목으로 분리되는 오류. 단/봉지/팩/병/통/박스 앞 숫자를 수량으로 인식하도록 파싱 수정.

5. "ㅇㅇ" 같은 한글 자모 단독 입력이 품목으로 인식되어 진행되는 버그. 자모만 있는 입력은 차단.

6. "이번 주 많이 쓴 재료 뭐야?" 같은 질문형 문장이 품목으로 인식되는 버그. 물음표로 끝나거나 이번/왜/어떻게/무슨/얼마나 등 질문 패턴은 LLM fallback으로 처리.
