index.html 수정 4개 진행해줘.

1. 입력창 왼쪽 마이크 버튼 추가
Web Speech API 사용. 마이크 버튼 클릭 시 음성 인식 시작, 인식된 텍스트를 입력창에 자동 입력.
버튼은 입력창 왼쪽 안쪽에 위치. 녹음 중 버튼 색상 변경 (빨간색).
한국어(ko-KR) 인식.

2. 확실한 파싱은 확인 카드 생략
아래 조건을 모두 만족하면 확인 카드 없이 바로 처리 후 채팅 결과만 표시:
- 품목명이 기존 등록 품목과 정확히 일치
- 수량이 명확하게 파싱됨
- 단일 액션 (입고/차감/재고설정 중 하나)
- 수량 이상값 하네스 미발동 (20 미만)
- 동일 품목 재입력 없음

위 조건 중 하나라도 불만족 시 기존대로 확인 카드 표시.

3. 재고 탭 수정 완료 시 결과 메시지 표시
재고 탭에서 품목 수정(수량/이름 등) 완료 시 채팅 영역에 결과 메시지 표시.
예: "하드롤이 3박스로 수정되었습니다"

4. 설정 변경 명령 처리 방식 수정
값이 명확한 설정 명령 (예: "냉동가리비 거래처 식봄으로 설정") → 확인 카드 없이 바로 처리.
처리 후 토스트 메시지로 "냉동가리비 거래처가 식봄으로 설정되었습니다" 표시 후 2초 후 사라짐.

값이 없는 명령 (예: "냉동가리비 거래처 바꿔줘") → 기존 선택 카드 표시 유지.
